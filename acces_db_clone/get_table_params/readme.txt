This tool Generates a table that contains specification data from all acces db columns
inserted at acc_file folder.

You can run this app with one or more .mdb or .accdb files and each file will return
a different table.

The following steps must be satisfied to get the expected result:

    1. Insert acces files onto acc_file (.\acc_file) folder
    2. Run run.py file located at actual folder (.\)
    3. Once step 2 is succesfully satisfied you can acces to the generated tables at tables (.\tables) folder

You can also find .py scripts at scripts (.\scripts) folder for any additional use of this tool
